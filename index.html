Introduction

Roadkill .NET is a lightweight but powerful Wiki engine built on the following foundations:

  • .NET 4.0
  • jQuery
  • ASP.NET MVC 3 with Razor
  • Lucene.net search engine
  • Creole, Media Wiki and Markdown syntax support
  • Bootstrap-based UI.
  • Supports SQL Server, SQL Server CE, Sqlite, MySQL, Postgres, MongoDB
  • It's themeable, well documented, supports Active Directory authentication and is (I hope) extremely easy to use. It's Free Open Source (FOSS)

Roadkill comes with a Screwturn import tool (Settings->Tools). This imports all pages, history, categories/tags and images as well.

Contributing

Currently looking for people to contribute language files (around 500 words and short phrases), for more information please ask on the Google Groups forum.

C# contributions are also welcome, but require a fair amount of experience with NHibernate, ASP.NET MVC/Razor or jQuery/HTML (pick two).

Please use Bitbucket to provide patches/pull requests/clones (Codeplex is a mirror repository)
